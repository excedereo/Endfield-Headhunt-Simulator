# Endfield Headhunt Simulator

Симулятор и калькулятор системы найма (gacha) **Arknights: Endfield** — баннер специального найма с rate-up 6★ оператором.

Статический сайт без бэкенда, вся логика на чистом JS, работает в браузере.

## Возможности

**Симулятор найма**
- **Одна крутка** — один заход на баннер, лог каждого 6★ и жетона
- **Монте-Карло** — тысячи прогонов: шанс получить копию, среднее, распределение потенциалов E0–E5
- **Сколько до цели** — сколько пуллов нужно до нужного потенциала (среднее/медиана/гистограмма)

Тяжёлые симуляции (до 300 000 прогонов) считаются в Web Worker — интерфейс не виснет.

**Калькулятор ресурсов**
- Подсчёт доступных пуллов из всех источников: вход, талоны АПК, базовые талоны, ороберил, ориджеметрий, донаты
- Чекаут с реальными деньгами за донаты
- Сохранение выбора между сессиями (localStorage)

## Механика баннера

| Редкость | Шанс |
|----------|------|
| 6★ | 0.8% |
| 5★ | 8% |
| 4★ | 91.2% |

- Софт-пити: после 65 пуллов без 6★ шанс растёт на +5% за пулл
- Хард-пити: 80-й пулл гарантирует 6★
- Rate-up: 50% что 6★ — это баннерный оператор; гарант на 120-м пулле (раз на баннер)
- Жетоны (потенциалы): за каждый дубликат rate-up + каждый 240-й пулл; макс E5

## Запуск локально

Web Worker требует http(s), поэтому открывать через локальный сервер:

```
python -m http.server 8000
```

затем открыть `http://localhost:8000`.
